# Onyx - Property Management

Onyx is a modern, AI-powered property management platform built with Django and React.

## Features

- Property portfolio management
- Tenant and lease management
- Online rent collection and payment history
- Maintenance request tracking
- In-app notifications and internal messaging
- Dashboard analytics and occupancy insights

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React, Material UI |
| Backend | Django, Django REST Framework |
| Database | SQLite (default), configurable |
| Payments | Stripe |
